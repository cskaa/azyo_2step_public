Initializing azyo scripts


create an instance of AzyoViewPort class and initialize it with the following:

root: the div where the azyo content will be loaded
client_code: 16 digit client code provided by azyo
user_name: the user identifier for which the azyo results will be generated
meta: User meta data which includes country, state and document_type for the user
on_finish: function that will be called once the results are generated 

    <script>
        const root = document.getElementById('exampleModal')
        
        AV = new AzyoViewPort(root, 
        client_code=`0000111100001111`, user_name=`test user 6`,
        meta = {'country': `India`, 'state': `Maharashtra`, 'document_type': `Driving License`},
        on_finish = (results) => {
            console.log('Results Obtained')
            console.log(results)
            
            $.ajax({
                url: '', // where ever you want to send the data, 
                data: JSON.stringify(results),
                type: "POST",
                dataType: "json",
                contentType: "application/json",
                processData: false,
                success: res => {},
                async: false,
                error: err => console.error('err')
            });

            console.log('You can Do Anything With these Results')
        })

        
    </script>

The Results

Each time AzyoViewport object is initialized it checks wather the results for the respective users are already generated. If results are generated it calls on_finish as the indicator for result status.
Results object received by on_finish method contains 2 things:
match precentage: precentage match between selfie image and the document image.
selfie_image: base64 encoded selfie image of the user. 

results = {
    match_precentage: 63.91216375197811,
    selfie_image: "data:image/png;base64,iVBORw0KGgoAAAANSUhE...
}
